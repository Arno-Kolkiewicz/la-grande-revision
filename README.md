# La Grande Révision

## Installation

- Cloner le repository
- Ouvrir le dossier avec Visual Studio Code
- Ouvrir un terminal dans Visual Studio Code
- Lancer la commande

```
npm install
```

- Lancer la commande

```
npm run dev
```

## Consigne

Réalisez chaque exercice en suivant les instructions dans la colonne de droite
